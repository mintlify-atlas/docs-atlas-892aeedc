## Overview

`NavHeader` is a self-contained, client-side navigation component that renders at the top of every page. It is fixed to the viewport (`position: fixed`, `z-50`) so it stays visible while the user scrolls.

**Source file:** `components/nav-header.tsx`

## Import

```typescript
import { NavHeader } from "@/components/nav-header"
```

## Props

`NavHeader` accepts no props. All navigation links and state are managed internally.

## Navigation Links

The component renders exactly six links, defined in the `navLinks` array at the top of the file:

```typescript
const navLinks = [
  { href: "/",                          label: "CIDTES" },
  { href: "/capacitacion",              label: "Capacitación" },
  { href: "/certificacion",             label: "Certificación" },
  { href: "/consultoria",               label: "Consultoría" },
  { href: "/investigacion-consultoria", label: "Investigación y Consultoría" },
  { href: "/cursos",                    label: "Cursos" },
]
```

## Usage

Mount `NavHeader` once, at the top of any page layout. Because pages use `pt-16` (or `pt-20`) to offset the fixed bar, add the same padding to the `<main>` element:

```typescript
import { NavHeader } from "@/components/nav-header"

export default function MyPage() {
  return (
    <div className="min-h-screen bg-white">
      <NavHeader />
      <main className="pt-16">
        {/* page content */}
      </main>
    </div>
  )
}
```

## Responsive Behaviour

<Tabs>
  <Tab title="Desktop (md and above)">
    All six links are rendered in a horizontal `<nav>` element. Links are separated by a `|` divider character. The nav is hidden on small screens via `hidden md:flex`.

    **Hover gradient effect:** each link has two absolutely-positioned gradient layers that animate in on hover:

    - A primary layer that **slides in** from left to right (`-translate-x-full` → `translate-x-0`) over 500 ms
    - A blurred secondary layer that **scales up** (`scale-0` → `scale-110`) over 700 ms

    Both layers use the same `from-[#6B46C1] via-[#0891B2] to-[#059669]` diagonal gradient (purple → cyan → emerald). The link text turns white on hover via `hover:text-white`.

    The link also scales up slightly (`hover:scale-105`) and gains a larger box shadow on hover.
  </Tab>
  <Tab title="Mobile (below md)">
    The desktop nav is hidden. A hamburger button (`aria-label="Menú"`) appears in its place. Clicking it toggles the `open` state:

    ```typescript
    const [open, setOpen] = useState(false)
    ```

    When `open` is `true`, a dropdown panel renders below the header bar with all six links stacked vertically. Each link applies the same purple–cyan–emerald gradient as a `hover:bg-gradient-to-tr` background.

    The hamburger icon itself animates: the top bar rotates `+45°`, the middle bar fades out, and the bottom bar rotates `−45°`, forming an `✕`. Clicking any dropdown link calls `setOpen(false)` to close the menu.
  </Tab>
</Tabs>
