## Overview

`DiagramaCircular` renders a 450 × 450 SVG wheel divided into three inner sectors (Seguridad e Higiene, Sustentabilidad, Energía) surrounded by five interactive outer band tabs. Clicking a tab rotates the wheel so that tab aligns to the top and fires the `onTabSelect` callback with the tab's identifier.

**Source file:** `components/diagrama-circular.tsx`

## Import

```typescript
import DiagramaCircular from "@/components/diagrama-circular"
```

## Props

<ParamField path="onTabSelect" type="(tabId: string | null) => void" optional>
  Callback fired when the user clicks an outer band tab. Receives the tab's `id` string (see table below), or `null` if the selection is cleared. Used by the parent page to display contextual service content beside the diagram.
</ParamField>

## Tab Definitions

The five outer bands are defined in the `tabs` array. Each has a fixed angular position and colour:

| `id` | Label | Colour | Angle (°) | Span (°) |
|---|---|---|---|---|
| `capacitacion` | Capacitación | `#00AEEF` | 2 | 50 |
| `estudios` | Estudios e Investigación | `#005B9A` | 95 | 130 |
| `consultoria` | Consultoría | `#4FC3F7` | 186 | 45 |
| `proyectos` | Proyectos | `#A9DDFF` | 238 | 50 |
| `certificacion` | Certificación | `#4FC3F7` | 299 | 70 |

<Info>
  Angles use a clock convention: 0° is up, increasing clockwise. The `layoutTabsWithEqualGaps` function redistributes the tabs so inter-band gaps are exactly equal (≈3° each), anchoring the `capacitacion` tab to preserve its original central angle.
</Info>

## Inner Sector Definitions

The three inner sectors are rendered as SVG pie slices, each filling 120° of the wheel:

| `id` | Label | Angular Range | Fill Gradient |
|---|---|---|---|
| `seguridad` | Seguridad e Higiene | 0° – 120° | `seguridadGradient` (cyan blues) |
| `sustentabilidad` | Sustentabilidad | 120° – 240° | `sustentabilidadGradient` (dark blues) |
| `energia` | Energía | 240° – 360° | `energiaGradient` (light blues / near-white) |

Each sector displays an inline SVG icon and a text label positioned radially using `SECTOR_LAYOUT` offsets (`iconRadius`, `textRadius`, `iconSize`, `fontSize`).

## Interaction Model

The component uses a single state variable:

```typescript
const [selectedTab, setSelectedTab] = useState<Tab | null>(null)
```

When a tab is clicked, `handleTabClick` sets `selectedTab` and calls `onTabSelect?.(tab.id)`. The entire outer band `<g>` group then rotates to align the selected tab to the top:

```typescript
const rotation = selectedTab ? -selectedTab.angle : 0

// Applied via inline style on the outer band group:
style={{
  transformOrigin: `${CX}px ${CY}px`,
  transform: `rotate(${rotation}deg)`,
  transition: "transform 900ms cubic-bezier(0.22, 1, 0.36, 1)",
}}
```

The inner three sectors and the centre badge do **not** rotate — only the outer band layer animates.

## Visual Layers (draw order)

1. **White ring** — `circle` at `r=180`, acts as a border between inner wheel and outer bands
2. **Three inner sector paths** — filled with radial gradients
3. **Sector icons and text** — SVG paths + `<text>` elements placed radially
4. **Outer band tabs** — animated `<g>` group; each tab is a rounded arc path (`roundedBandPath`)
5. **Curved tab labels** — `<textPath>` following the midline arc of each band
6. **Centre badge** — white `circle` at `r=52` with the word "Áreas"

## Usage Example

This is how the component is used on `app/page.tsx`:

```typescript
"use client"

import { useState } from "react"
import DiagramaCircular from "@/components/diagrama-circular"

export default function HomePage() {
  const [selectedService, setSelectedService] = useState<string | null>(null)

  const serviceContent: Record<string, { title: string; description: string[] }> = {
    capacitacion: {
      title: "Capacitación",
      description: [
        "Ofrecemos cursos y talleres especializados en energía, sustentabilidad y seguridad laboral.",
        "Nuestros programas están diseñados para desarrollar competencias técnicas y profesionales.",
      ],
    },
    // ... estudios, consultoria, proyectos, certificacion
  }

  return (
    <div className="grid lg:grid-cols-2 gap-16 items-center">
      {/* Diagram on the left */}
      <DiagramaCircular onTabSelect={setSelectedService} />

      {/* Service description on the right */}
      <div>
        <h2>{selectedService ? serviceContent[selectedService].title : "Tipo de Servicio"}</h2>
        {selectedService &&
          serviceContent[selectedService].description.map((p, i) => (
            <p key={i}>{p}</p>
          ))}
      </div>
    </div>
  )
}
```

## SVG Effects

<AccordionGroup>
  <Accordion title="Drop shadow on hover">
    Each tab path uses `onMouseEnter`/`onMouseLeave` handlers to increase the `drop-shadow` filter intensity, giving a raised appearance. The active (selected) tab gets a stronger shadow plus `brightness(1.12)` and the `innerGlow` SVG filter.
  </Accordion>
  <Accordion title="Overall diagram shadow">
    The root `<svg>` element has `filter: drop-shadow(0 20px 40px rgba(0,0,0,0.25))` applied via inline style, giving the whole wheel a floating appearance against the page background.
  </Accordion>
  <Accordion title="Gradients">
    Every inner sector and outer band has its own named `<linearGradient>` or `<radialGradient>` defined in `<defs>`. This keeps colours consistent even when the SVG is scaled.
  </Accordion>
</AccordionGroup>
